# Glossary

**Artifact**
A stored output produced by a pipeline step, such as transcript JSON or a combined feature table.

**Extension**
A pluggable unit that performs one pipeline step (for example, VOC structured extraction).

**Feature-wide table**
A single table with one row per call and columns for all extracted features.

**Pipeline**
A config-driven sequence or graph of steps with conditional branching.

**Prompt template**
System and user instructions used to build the LLM request for a step.

**VOC**
Voice of Customer pipeline that extracts renewal intent and issue details from calls.
